PASOS PARA VISUALIZAR PROYECTO

1. Abrir XAMPP y encender Apache y MySQL.
2. Importar el archivo base.sql a phpMyAdmin para crear la base de datos necesaria para el proyecto.
3. Clonar el repositorio utilizando el enlace proporcionado de GitHub / descargar la carpeta como zip.
4. Copiar los archivos en la carpeta htdocs de XAMPP.
5. Abrir un navegador web y visitar:
   - Si descargaste el proyecto como zip: http://localhost/NexlanFinal-main
   - Si clonaste el repositorio: http://localhost/NexlanFinal
